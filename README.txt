Codigo de test de compatibilidad para usuarios que asisten a eventos de SpeedDating, el archivo TestCompatibilidadGrafico, 
contiene el código de el test de compatibilidad en el cual puedes introducir varios usuarios y posteriormente escoges a cuales quieres calcular
la compatibilidad, esta hecho con la libreria tkinter de python, esta versión no me dio tiempo a mostrarla en clase. 

El archivo TestCompatibilidadConsola es la primera versión mostrada en clase, la cual és el test hecho en consola y que como mucho puedes 
introducir dos usuarios diferentes.
